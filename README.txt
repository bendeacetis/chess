How to compile and run the program:
1. To compile and run, navigate to the directory in the terminal in which the files are stored (using cd commands).
2. Assuming you are in the correct directory, to compile, type in "javac Game.java".
3. To run the program, type in "java Game".

Assumptions:
- Assume that the user enters the rows/columns in the correct format.
- User knows how to play chess.
- If the wrong entry is inputted, the move is skipped (nothing will be moved).

Known bugs or defects:
- White pieces are outlined, and black are solids (dark mode error).
